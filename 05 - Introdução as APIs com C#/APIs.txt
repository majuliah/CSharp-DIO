	🎃Introdução as APIs
		🤔O que é uma API?
		   Uma API (Application Programming Interface) é uma forma de comunicação entre computadores ou programas de computadores. É um software que fornece 			   informação para outro software. É uma forma de integração entre sistemas. 

		🧐 Exemplos de API's:
		   https://date.nager.at/ -> feriados ao redor do mundo
		   https://dog.ceo/dog-api/ -> cachorres
		
		🤓Aspectos importantes de uma API:
		  Ler a documentação é importante para sabermos os parâmetros necessários para realizar requisições.
